# GSR Images

Screenshots and assets for documentation. Organized by context.

| Folder | Contents |
|--------|----------|
| `icon/` | Mod icon, logo (`gsr_icon.png`, `gsr.png`) |
| `menu/` | Main menu, GSR Controls screenshots |
| `timer/` | Timer HUD: prestart, run started, splits (Nether, Bastion, End) |
| `victory_fail/` | Victory and fail broadcast screenshots |
| `locators/` | Locator HUD examples (Bastion, Stronghold) |
| `run_history/` | Run History screens: Run Info, Run Graph, Player Graph |
| `config/` | Config screens, Keybinds |

**Note:** Mod runtime assets (`assets/gsr/`, `assets/minecraft/textures/gsr/`) remain in `src/main/resources` for Fabric resource loading.
